## Instructions
* Download the "address-book.tar" file
* cd into downloaded folder <br>
` cd address-book `
* Run following command to extract make it ready to run

` tar -xvf address-book.tar `
<br>
` cd address-book `
<br>
` chmod +x main ` <br>
* In the same directory run following command to run the file
` ./main `
